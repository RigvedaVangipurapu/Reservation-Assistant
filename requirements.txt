# Storage: 100MB for dependencies, 500MB for browser binaries

# Installation Instructions
# =========================
# 1. Create virtual environment:
#    python3 -m venv badminton_agent_env
#    source badminton_agent_env/bin/activate  # On Windows: badminton_agent_env\Scripts\activate
#
# 2. Install dependencies:
#    pip install -r requirements.txt
#
# 3. Install Playwright browsers:
#    playwright install chromium
#
# 4. Set up environment variables:
#    export GOOGLE_API_KEY="your_api_key_here"
#
# 5. Run the application:
#    streamlit run app.py

# Version Compatibility
# =====================
# This requirements.txt is specifically tested with:

# - Playwright 1.40.0
# - Google Generative AI 0.3.0
